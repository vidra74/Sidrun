README

Sidrun is mobile application for Red Bull Sidrun event: http://www.redbull.com/hr/hr/events/1331663829610/red-bull-sidrun-2014.

User of Sidrun application is organizer or participant of event. Appliaction provides info and messages from organizers to participants in case of change.
